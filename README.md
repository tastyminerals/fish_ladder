# fish_ladder
Simple console client to check BW ladder stats on Korean "Fish" server.

### Usage
`python2 fish_ladder.py -h` or `./fish_ladder.py -h`

### Examples
`./fish_ladder.py`

![0](http://i.imgur.com/i5rDwyh.png)

`./fish_ladder.py -t 5`

![1](http://i.imgur.com/YXx4nsi.png)
